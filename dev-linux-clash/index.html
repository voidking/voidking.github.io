Clash: A rule-based tunnel in Go.

Clash是一个非常流行的科学上网客户端，适用于windows、macos、linux、android。

本文中，我们学习在linux中安装配置clash。其他系统平台都是图形化界面操作，不用额外学习。

2. 安装Clash

访问Clash Releases，下载linux平台的软件包，这里选择下载clash-linux-amd64-v1.14.0.gz

2、安装Clash

3、初始化

Clash 运行时需要 Country.mmdb 文件，当第一次启动 Clash 时，会自动下载 Country.mmdb 到 $HOME/.config/clash 目录下。
如果下载卡住，建议手动下载Country.mmdb放到 $HOME/.config/clash ，然后重新执行clash。

当看到以下信息，表明初始化成功，ctrl+C退出即可。
INFO[0204] Mixed(http+socks) proxy listening at: 127.0.0.1:7890

注：Country.mmdb 文件利用 GeoIP2 服务能识别互联网用户的地点位置，以供规则分流时使用。

5. Web GUI管理

5.1. 局域网管理

Clash 提供了默认的 9090 端口作为GUI管理端口。

1、测试管理端口

2、访问管理界面
访问Clash外部控制，填入Clash的配置信息。

  • Host：填入Clash所在主机的局域网IP或者局域网域名，本机要和Clash所在主机连通
  • 端口：7890
  • 密钥：密钥配置在 /etc/clash/config.yaml 文件中，字段为secret，默认为空。

依次点击 添加，确定，就可以对Clash进行配置了。

3、退出管理页面
Clash的Web GUI默认是没有退出选项的，要想退出，可以手动清除下 Local Storage 中的缓存。
具体操作：F12打开调试工具，Application，Storage，Local Storage，选中网址，右键，Clear。
