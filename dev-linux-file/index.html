1. 前言

本文记录Linux中常用的文件管理命令，主要包括：

  • cp：拷贝文件
  • mv：移动文件
  • rm：删除文件
  • tar：解压与压缩文件
  • zip：解压与压缩文件
  • unar：解压文件
  • split：拆分文件
