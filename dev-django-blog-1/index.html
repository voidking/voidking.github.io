后台管理

django自带后台管理系统Admin，被授权的用户可以直接在后台管理系统中操作数据库。同时，我们可以按照需求对Admin进行定制。

1、创建超级用户
python manage.py createsuperuser
根据提示，输入用户名、邮箱、密码。

2、测试访问
访问地址 http://localhost:8000/admin/ ，输入用户名密码，登录后台管理系统。
