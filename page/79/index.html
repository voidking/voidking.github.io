好好学Linux：编码格式转换

发表于 2017-06-02 更新于 2022-12-12 分类于 engineering ， linux ， shell

编码格式修改

执行sh脚本时报错：/bin/sh^M: bad interpreter: No such file or directory

1、添加执行权限
chmod a+x filename

2、然后修改文件格式
vi filename

3、利用如下命令查看文件格式

:set ff 或 :set fileformat

可以看到 fileformat=dos 或 fileformat=unix。

4、利用如下命令修改文件格式

:set ff=unix 或 :set fileformat=unix

5、保存文件
:wq

6、最后再执行文件
./filename
