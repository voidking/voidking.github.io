设置项目默认编码

在eclipse中，Window，Preference，General，Workspace，Text file encoding，Other选择UTF-8。

设置JSP页面默认编码

在eclipse中，Window，Preference，Web，JSP Files，Encoding选择ISO 10646/Uincode(UTF-8)。

设置代码提示

在eclipse中，Window，Preference，Java，Editor，Content Assist，Auto Activation，Auto activation triggers for Java中输入“.”和52个英文字符。

在eclipse中，Window，Preference，General，Keys。

大学生十八条受益一生的定律

发表于 2014-01-18 分类于 爱好 ， 读书

　　
1、一定要有独立的人格、独立的思想。一个经过独立思考而坚持错误观点的人比一个不假思索而接受正确观点的人更值得肯定。不要成为灌输教育的牺牲品。
　　
2、仕途，商界，学术。大致说来，每个人都注定要走上三条道路中的某一条。在进行职业生涯规划的时候，不妨以此作为思考的出发点。根据不同的职业生涯规划来塑造各自的核心竞争力。只有知道自己以后要做什么，才能知道自己应该学什么。
　　
3、专业无冷热，学校无高低。没有哪个用人单位会认为你代表了你的学校或者你的专业。千万不要因为你是名牌大学或者热门专业而沾沾自喜，也大可不必因为你的学校不好或者专业冷门而自卑。
