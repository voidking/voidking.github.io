yum命令

发表于 2013-10-21 分类于 engineering ， devops

通过yum安装和删除RPM包

安装rpm包,如dhcp
yum install dhcp

删除rpm包,包括与该包有依赖性的包
yum remove licq
注意:同时会提示删除licq-gnome,licq-qt,licq-text

通过yum工具更新软件包

检查可更新的rpm包：
yum check-update

更新所有的rpm包：
yum update
